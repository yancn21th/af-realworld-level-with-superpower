Design Section 1: Architecture Overview

Full-stack RealWorld — Hono + React + Drizzle + SQLite — Monorepo

Monorepo Structure

Project Layout
realworld-app/
├── apps/
│   ├── api/                 ← Hono backend
│   │   ├── src/
│   │   │   ├── routes/      ← /users, /articles, /profiles, etc.
│   │   │   ├── middleware/  ← JWT auth, error handler
│   │   │   ├── db/          ← Drizzle schema + migrations
│   │   │   └── index.ts     ← Entry point
│   │   └── package.json
│   └── web/                 ← React + Vite SPA
│       ├── src/
│       │   ├── pages/       ← Home, Login, Editor, Profile...
│       │   ├── components/  ← ArticleList, Header, Footer...
│       │   ├── api/         ← API client (fetch wrappers)
│       │   └── main.tsx
│       └── package.json
├── packages/
│   └── shared/              ← Shared TypeScript types
│       └── src/types.ts
├── specs/                   ← (symlink or copy of realworld specs)
└── package.json             ← pnpm workspace root
